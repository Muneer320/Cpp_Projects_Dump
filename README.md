## 🛠 Build & Run

Each project is a standalone `.cpp` file. Compile and run:

```bash
g++ "Snake Game/Snake Game.cpp" -o snake
./snake
```

Or use any C++ compiler of your choice (MSVC, Clang, etc.).

## 🎯 Why This Exists

This was my C++ sandbox. When I was learning:
- Pointers and memory management
- Object-oriented programming in C++
- Console input/output and basic game loops
- File handling for management systems

Every project here represents a concept clicked into place. They're kept here as a time capsule — a reminder that everyone starts somewhere.
